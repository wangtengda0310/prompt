# 后台管理系统 页面布局

> 文件路径: `frontend/src/pages/Admin.vue`
> 路由: `/admin`

## 概述

典型的后台管理系统主页面，采用经典的三段式布局，左侧为功能菜单树，右侧为 Tab 内容面板，顶部为操作工具栏，底部为系统状态栏。适用于需要多模块切换和多标签页操作的复杂管理场景。

## 布局尺寸

| 区域 | 尺寸 | 说明 |
|------|------|------|
| Header (工具栏) | 高度 56px | 固定高度，包含操作按钮和搜索框 |
| Footer (状态栏) | 高度 32px | 固定高度，显示系统状态信息 |
| 左侧 Sider | 宽度 220px (展开) / 64px (折叠) | 可折叠的菜单侧边栏 |
| Tab 栏 | 高度 40px | 标签页导航区域 |
| 内容区 | 自适应 | 填充剩余空间，内容过多时可滚动 |

## 数据流

```
用户点击菜单项
    │
    ▼
handleMenuClick(menuItem)
    │
    ├──► 判断是否已打开对应 Tab
    │         │
    │         ├── 已存在 → 激活该 Tab
    │         │
    │         └── 不存在 → 创建新 Tab 并激活
    │                   │
    │                   ▼
    │              tabList.push(newTab)
    │                   │
    └───────────────────┴──► activeTabKey 更新
                               │
                               └──► TabPanel 组件响应式更新显示内容

用户点击 Tab 关闭按钮
    │
    ▼
handleTabClose(tabKey)
    │
    ├──► 从 tabList 移除该 Tab
    │
    └──► 如果关闭的是当前激活 Tab → 激活相邻 Tab
```

## 关键状态

| 状态 | 文件 | 类型 | 说明 |
|------|------|------|------|
| menuCollapsed | stores/layout.ts | Ref\<boolean\> | 菜单折叠状态 |
| menuList | stores/menu.ts | Ref\<MenuItem[]\> | 菜单树数据 |
| tabList | stores/tabs.ts | Ref\<TabItem[]\> | 已打开的 Tab 列表 |
| activeTabKey | stores/tabs.ts | Ref\<string\> | 当前激活的 Tab |
| toolBarLoading | stores/toolbar.ts | Ref\<boolean\> | 工具栏按钮加载状态 |

## 交互说明

| 操作 | 触发 | 说明 |
|------|------|------|
| 点击菜单项 | handleMenuClick() | 打开或切换到对应的 Tab 页 |
| 点击 Tab | handleTabChange() | 切换到指定的 Tab 内容 |
| 点击 Tab 关闭按钮 | handleTabClose() | 关闭指定的 Tab |
| 点击折叠按钮 | toggleMenuCollapse() | 切换菜单展开/折叠状态 |
| 点击工具栏按钮 | handleToolAction() | 执行对应的工具操作（新建/编辑/删除等） |
| 搜索框输入 | handleSearch() | 触发搜索功能，支持防抖 |
| 键盘快捷键 | handleKeydown() | 支持 Ctrl+W 关闭当前 Tab 等快捷键 |

## 响应式设计

| 屏幕宽度 | 布局变化 |
|---------|---------|
| > 1200px | 完整布局，菜单展开 |
| 768px - 1200px | 菜单默认折叠，Tab 标签缩短 |
| < 768px | 菜单隐藏（通过按钮呼出），Tab 改为下拉选择 |
