# 后台管理系统页面布局说明文档

## 整体结构

```
+----------------------------------------------------------+
|                        工具栏 (顶部)                        |
+------------+---------------------------------------------+
|            |                                             |
|            |              Tab 面板区域                     |
|   菜单树    |                                             |
|   (左侧)   |              (右侧主内容区)                   |
|            |                                             |
|            |                                             |
+------------+---------------------------------------------+
|                        状态栏 (底部)                        |
+----------------------------------------------------------+
```

## 布局组件详解

### 1. 工具栏 (Toolbar)

**位置**: 页面顶部，固定高度

**功能**:
- 系统 Logo 和标题
- 全局搜索框
- 用户信息/头像
- 通知图标
- 系统设置入口
- 退出登录按钮

**推荐高度**: 48px - 64px

**技术实现建议**:
- 使用 `position: fixed` 或 `position: sticky` 固定在顶部
- 采用 Flexbox 布局，左右分区

### 2. 菜单树 (Sidebar Menu)

**位置**: 页面左侧，固定宽度或可折叠

**功能**:
- 多级菜单导航
- 菜单展开/折叠
- 当前选中状态高亮
- 支持 icon + 文字显示
- 可选: 收起为 icon 模式

**推荐宽度**:
- 展开状态: 200px - 260px
- 折叠状态: 48px - 64px

**技术实现建议**:
- 使用递归组件实现多级菜单
- 支持 `collapsed` 状态切换
- 菜单项使用 `router-link` 实现路由跳转

### 3. Tab 面板区域 (Main Content)

**位置**: 页面右侧主区域

**功能**:
- 多 Tab 页签切换
- Tab 关闭功能
- Tab 拖拽排序 (可选)
- 右键菜单: 关闭其他/关闭所有/刷新
- 面包屑导航 (可选)

**布局结构**:
```
+----------------------------------+
|  Tab1  |  Tab2  |  Tab3  |  +    |  <- Tab 栏
+----------------------------------+
|                                  |
|         页面内容区域               |
|                                  |
+----------------------------------+
```

**技术实现建议**:
- Tab 数据存储在全局状态管理中
- 使用 `keep-alive` 缓存已打开的页面组件
- 监听路由变化自动添加 Tab

### 4. 状态栏 (Footer)

**位置**: 页面底部，固定高度

**功能**:
- 系统版本信息
- 当前时间显示
- 在线状态
- 操作提示信息
- 进度条显示 (可选)

**推荐高度**: 24px - 32px

**技术实现建议**:
- 使用 `position: fixed` 固定在底部
- 响应式适配不同屏幕宽度

## 响应式设计

### 断点设置

| 断点名称 | 宽度范围 | 布局调整 |
|---------|---------|---------|
| 大屏幕 | >= 1200px | 菜单展开，完整显示 |
| 中等屏幕 | 768px - 1199px | 菜单可折叠 |
| 小屏幕 | < 768px | 菜单隐藏，抽屉式弹出 |

### 移动端适配

- 菜单树转为抽屉组件 (Drawer)
- Tab 面板改为可滑动切换
- 工具栏简化显示

## CSS 布局方案

### 推荐方案: CSS Grid + Flexbox

```css
/* 整体布局 */
.app-layout {
  display: grid;
  grid-template-rows: 60px 1fr 32px;
  grid-template-columns: 220px 1fr;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  height: 100vh;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; overflow: auto; }
.footer { grid-area: footer; }
```

### 备选方案: Flexbox

```css
.app-layout {
  display: flex;
  flex-direction: column;
  height: 100vh;
}

.body-layout {
  display: flex;
  flex: 1;
  overflow: hidden;
}

.sidebar { width: 220px; flex-shrink: 0; }
.main { flex: 1; overflow: auto; }
```

## 组件选型建议

### Vue 3 + Naive UI

| 组件 | Naive UI 组件 |
|-----|--------------|
| 菜单树 | `n-menu` |
| Tab 面板 | `n-tabs` |
| 工具栏 | `n-layout-header` |
| 状态栏 | `n-layout-footer` |
| 整体布局 | `n-layout` |

### 状态管理

建议使用 Pinia 管理:
- 当前激活的菜单项
- 已打开的 Tab 列表
- 菜单折叠状态
- 用户信息

## 交互规范

### 菜单交互
- 点击菜单项: 跳转页面并添加 Tab
- 悬停: 显示 tooltip (折叠模式下)
- 当前页面: 高亮对应菜单项

### Tab 交互
- 点击 Tab: 切换页面
- 点击关闭按钮: 关闭 Tab
- 中键点击: 关闭 Tab
- 双击: 刷新当前页面 (可选)

### 快捷键
- `Ctrl + W`: 关闭当前 Tab
- `Ctrl + Tab`: 切换到下一个 Tab
- `Ctrl + 数字`: 切换到指定 Tab

## 注意事项

1. **性能优化**
   - 使用虚拟滚动处理大量菜单项
   - Tab 页面使用 `keep-alive` 缓存
   - 限制最大 Tab 数量

2. **无障碍访问**
   - 菜单支持键盘导航
   - 添加适当的 ARIA 属性
   - 焦点管理

3. **状态持久化**
   - Tab 状态保存到 localStorage
   - 页面刷新后恢复 Tab 状态
